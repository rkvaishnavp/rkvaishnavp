Hardware and Software Engineer
------------------------------

Hello, I'm Vaishnav Prasad Ravilichitty Kishor, a current B.E. student majoring in Electronics and Instrumentation at BITS Pilani, K.K. Birla Goa Campus. My technical skills include programming in C++, Verilog, C, JavaScript, Python, and proficiency in tools like Xilinx Vivado, Git, and Postman. I have hands-on experience in full-stack web development, evident in projects like Movie-Mingle Streaming Website and Threads Clone. In kernel programming, I added a custom system call to Linux Kernel 5.4.0, and in device driver development, I created a driver adjusting screen brightness based on laptop charging status. My digital design and Computer Architecture projects include a RISC-V CPU Core and Handwritten Digit Recognition in FPGA.
